# Stewbot
This is the open source repository for Stewbot, an advanced Discord bot with the intention of helping keep your server running smoothly while also providing entertainment.
Stewbot is completely open source and free.

## Running Locally
Want to run your own copy? Follow these steps:
1. Have the latest LTS node installed
2. Copy the contents of `./example.env.jsonc` to a new file named `env.json` this root directory
3. Remove the comments and fill in the keys
4. Run `npm install` to install dependencies
5. Run `node launchCommands.js`
6. Start the bot with `node index.js`
